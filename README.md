# Notebook

Digital journaling app inspired by a personal notebook system. Focused on daily entries, habit building, reminders, and structured reflection. Built as a learning project to turn real-life workflows into scalable software.


## 📓 Digital Notebook & Daily Journaling System

This project is an attempt to transform my personal handwritten notebook system into a structured, professional digital product.

The goal is to design and build a simple yet powerful journaling application that helps users organize their daily thoughts, track habits, and build consistency through a clean and intuitive interface.

### 🚀 Purpose

I started this project to:

* Learn how to properly **start and structure software projects**
* Turn a **manual system (paper notebook)** into a scalable digital solution
* Build a tool that encourages **daily reflection and consistency**

### 🧠 Core Idea

The application revolves around a **daily journaling workflow**, where users can:

* Write entries for each day from a dedicated home page
* View and navigate all past entries in a structured way
* Build a habit of daily writing through reminders and simple UX

### ✨ Planned Features

* 🏠 **Home Page Dashboard**
  * Daily input field for quick journaling
  * Focus on simplicity and speed
* 📅 **Entries History Page**
  * View all previous journal entries
  * Organized by date for easy navigation
* ⏰ **Notification System**
  * Daily reminders to write entries
  * Customizable time settings
* 📱 **Screen Time Tracking (Planned)**
  * Automatic integration (if possible)
  * Reduce manual input and improve accuracy
* 🎨 **UI/UX Design (Upcoming)**
  * Will be designed in Figma
  * Focus on clarity, minimalism, and usability

### 🔍 Vision

This is not just a journaling app — it's a learning project to understand:

* How to go from an idea → structured product
* How to design systems based on real-life workflows
* How to build scalable and user-focused applications
